## 💻 Project Description
This project is a **Library Management System** developed using Java to automate basic library operations. It helps manage book records efficiently by allowing users to add, view, issue, and return books.

The system replaces manual record-keeping with a simple digital solution, reducing errors and saving time.

---

## 🎯 Objectives
- Automate library operations  
- Manage book records efficiently  
- Implement Java OOP concepts  
- Build a simple and user-friendly system  

---

## ⚙️ Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- ArrayList
- Command Line Interface (CLI)

---

## ✨ Features
- 📚 Add new books  
- 📖 View all books  
- 🔄 Issue books  
- ✅ Return books  
- 🔍 Check availability
